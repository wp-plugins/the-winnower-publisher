Publish, peer review and get cited with your own DOI by cross posting to thewinnower.com

== Description ==
This plugin lets you post on both your website and thewinnower.com. When you publish on The Winnower you're able to take advantage of an open peer review processes, and publish your work with a DOI.

== Installation ==
This plugin requires an account with The Winnower and willingness to advance science. It also requires the Piklist plugin which allows for great plugin development.
